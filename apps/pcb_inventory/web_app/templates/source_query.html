SQL Reference & Examples
Common Access SQL Syntax
  • SELECT * FROM tblPCB_Inventory - All records
  • WHERE Qty > 100 - Numeric comparison
  • WHERE Job LIKE '*4187*' - Text search (wildcard)
  • WHERE Tran_Time >= #2024-01-01# - Date comparison
  • ORDER BY Qty DESC - Sort results
  • GROUP BY Job - Group records
Available Tables
  • tblPCB_Inventory - Main PCB inventory
  • tblTransaction - Transaction history
  • tblWhse_Inventory - Warehouse inventory
  • tblUser - User accounts
  • TranCode - Transaction codes
  • tblBOM - Bill of Materials
Sample Queries
SELECT Job, PCB_Type, SUM(Qty) as Total FROM tblPCB_Inventory GROUP BY Job, PCB_Type
Inventory summary by job and type
SELECT * FROM tblTransaction WHERE TranType = 'PICK' ORDER BY Tran_Time DESC
Recent pick transactions
Advanced Examples
SELECT t.Job, t.TranType, t.TranQty, p.PCB_Type FROM tblTransaction t INNER JOIN tblPCB_Inventory p ON t.Job = p.Job
Join transactions with inventory
SELECT TranType, COUNT(*) as Count, SUM(TranQty) as TotalQty FROM tblTransaction GROUP BY TranType
Transaction type statistics
